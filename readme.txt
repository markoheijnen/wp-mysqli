== Description ==

MySQL is deprecated and this plugin creates a layer to use MySQLi instead. So you don't get notices when using PHP 5.5

Be sure to read the installation instructions, as this is **not** a traditional plugin, and needs to be installed in a specific location.


== Installation ==

1. Verify that you have MySQLi
2. Copy `db.php` to your WordPress content directory (`wp-content/` by default).
3. Done!
